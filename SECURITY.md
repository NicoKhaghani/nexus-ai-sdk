# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability, please report it privately.

Email: security@nexus-ai.dev

Please include:

- description of the issue
- reproduction steps
- potential impact

We will investigate and respond as quickly as possible.

## Security Practices

Nexus follows several security principles:

- dependency isolation through package boundaries
- runtime capability sandboxing
- validation of intermediate execution outputs
- controlled capability routing
- deterministic execution graphs

## Responsible Disclosure

We ask security researchers to follow responsible disclosure and avoid public reporting until the issue is resolved.
